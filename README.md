# Aplicativo de Receitas: Navegação e Detalhes Completos

O que o aplicativo faz:

Esta etapa aprimorou o aplicativo com um visual profissional (usando React Native Paper) e navegação completa.

Exploração de Receitas: O usuário pode pesquisar receitas na lista.

Detalhes: Ao clicar em qualquer receita, o usuário é levado a uma nova tela que mostra as instruções completas, a lista de ingredientes e a área de origem do prato.

Vídeo: Se houver um vídeo de preparo disponível, um botão permite ao usuário assistir a receita no YouTube.
